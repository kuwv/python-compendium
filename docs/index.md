# Welcome to Compendium

This website provides the project documentation for Python Compendium.

## Overview

Compendium is a configuration management tool

## Features

- [ ] Configurations
    - [ ] Hierarchical configuration

- [ ] Filetypes:
    - [ ] JSON support
    - [ ] INI support
    - [ ] TOML support
    - [ ] XML support
    - [ ] YAML support
- [ ] Extensible
